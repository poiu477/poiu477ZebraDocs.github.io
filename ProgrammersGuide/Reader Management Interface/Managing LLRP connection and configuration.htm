Managing LLRP connection and configuration

The RFID Reader can be configured in LLRP client or server mode using the function rm.LLRPConnection.getLLRPConnectionConfig . While acting in client mode, the method rm.LLRPConnection.initiateFromReader can be used to intiate an LLRP Connection from the reader to the port on the Server-IP as configured in setLLRPConnectionConfig. rm.LLRPConnection.disconnectFromReader disconnects the current LLRP Connection from the reader.

 

try {

 

    // Get LLRP Connection Configuration

    LLRPConnectionConfig config = rm.LLRPConnection.getLLRPConnectionConfig();

 

    System.out.println("Get LLRP Connection");

    System.out.println("IsClient         : " + config.isClient());

    System.out.println("Port             : " + config.getPort());

    System.out.println("Host Server IP   : " + config.getHostServerIP());

 

    config.setClient(true);

 

    // Set LLRP Connection Configuration          

    rm.LLRPConnection.setLLRPConnectionConfig(config);

 

    // Initialate LLRP connection from the reader

    rm.LLRPConnection.initiateFromReader();

 

    …

 

    rm.LLRPConnection.disconnectFromReader();

 

 

 

} catch (OperationFailureException exception) {

    System.out.println("LLRPConnection.GetConfig failed " + exception.getStatusDescription() + " " + exception.getVendorMessage());

}
