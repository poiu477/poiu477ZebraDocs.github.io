RFID API - 3

The programmer’s guide is divided into two parts:

The first part pertains to the Generic Reader Interface which covers the API and Usage supported by the reader via the LLRP Interface. This interface is used for routine operations like Reading Tags (Inventory), Programming Tags (Tag Access Operations like Read, Write, Lock, Kill) and configuring the Reader.

The second part pertains to the Reader Management Interface which is used for managing the Reader (Software Update, Reboot, Using Config Files, etc).
